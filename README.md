# 🏥 Hospital

Proyecto desarrollado en **Java (Vanilla)** utilizando **Maven** como gestor de dependencias.

El objetivo del proyecto poner en practica la POO y los principios SOLID (principalmente).

---

## 📁 Estructura del proyecto

```
Hospital/
├── .idea/
├── .mvn/
├── src/
│   ├── main/
│   │   ├── java/
│   │   └── resources/
│   └── test/
│       └── java/
│           ├── MedicalTest.java
│           └── StudentMedicalTest.java
├── target/
├── .gitignore
├── pom.xml
└── README.md
```

---

## ⚙️ Tecnologías utilizadas

- Java
- Maven
- JUnit (para pruebas unitarias)

---

## 🧪 Testing

El proyecto incluye pruebas unitarias ubicadas en:

```
src/test/java
```

Para ejecutar los tests:

```bash
mvn test
```
